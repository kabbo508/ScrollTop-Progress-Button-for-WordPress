# ScrollTop Progress Button

A modern and lightweight WordPress plugin that adds a smooth **Back to Top button** with a **circular scroll progress indicator**, full customization, and live preview in admin.

---

## 🚀 Features

- Smooth scroll back-to-top button
- Circular **scroll progress indicator**
- Live preview inside admin panel
- Fully customizable:
  - Colors (background, icon, ring, track)
  - Button size
  - Position (left/right)
  - Offset spacing
- Font Awesome icon selection
- Custom icon class support
- Lightweight and performance optimized
- No jQuery dependency on frontend (clean JS)

---

## 🎨 Customization Options

- Enable / Disable button
- Position:
  - Bottom right
  - Bottom left
- Show after scroll (px)
- Button size (36px–96px)
- Colors:
  - Background
  - Icon
  - Progress ring
  - Ring track (supports RGBA)
- Icon:
  - Built-in selection
  - Custom Font Awesome class

---

## 🧠 How It Works

- Button appears after user scrolls defined distance
- Clicking button scrolls smoothly to top
- Progress ring visually represents scroll position
- Styles are dynamically injected using CSS variables
